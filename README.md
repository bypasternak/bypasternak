**Учебный проект** — сервис для публикации и хранения кулинарных рецептов.

**Стек:** Python • Django • Django REST Framework • PostgreSQL • Docker

**Основные возможности:**

- Авторизация пользователей
- Управление рецептами, ингредиентами и тегами
- Избранное и список покупок

### 🔗 YaCut

**Учебный проект** - асинхронный сервис для сокращения ссылок с возможностью создания коротких URL и последующего перенаправления на оригинальный адрес.

**Стек:** Python • FastAPI • SQLAlchemy • SQLite • Uvicorn

**Основные возможности:**
- Создание коротких ссылок
- Автоматическая и пользовательская генерация коротких идентификаторов
- Переадресация по короткой ссылке
- Валидация данных
- Асинхронная обработка запросов

**Репозиторий:** [Async YaCut](https://github.com/bypasternak/async-yacut)
